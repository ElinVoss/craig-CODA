def extract_section(text, start_heading):
    # extract from heading line to next ### heading
    pattern = re.escape("### "+start_heading)
    m=re.search(pattern, text)
    if not m: return ""
    start=m.start()
    # find next heading after start
    m2=re.search(r"^### ", text[m.end():], flags=re.M)
    end = m.end()+m2.start() if m2 else len(text)
    return text[start:end]

## 1. DOCUMENT PURPOSE

## 2. PROSE LAWS (NON-NEGOTIABLE)

### 2.1 Point of View

- Camera stays inside their sensory and conceptual limitsâ€”no omniscient commentary in main chapters.  

### 2.2 Tense

- Default: **tight past tense** for main chapters.  
- Short present-tense flashes allowed only for immediate, high-stress sensations (pain, impact, realization), and must be brief.

### 2.3 Language Constraints

- **No metaphors or similes** in main chapters.  
  - Do not write "like" / "as if" comparisons for imagery.  
  - Describe what is literally present: light, sound, motion, weight, temperature, stress.  
- Descriptions must be **plain, concrete, and physical**.  
  - "The beam bowed two fingers under the load" is correct.  
  - "The beam sagged like an old man" is not.

### 2.4 Magic Description (Grounded and Anime-Escalated)

- Soulpulse use is always described through **material effects and bodily cost**, not abstract energy:
  - Show beams straightening, glass knitting, joints locking, stone crumbling.  
  - Show shaking hands, shortness of breath, pain, exhaustion.  
- Even at late-saga anime levels:

- **Finger** (~2 cm)  
- **Hand** (~10 cm)  
- **Span** (~23 cm)

- **Stride** (~90 cm)  
- **Bar** (~1.8 m)

- **Cable** (~45 m)

- **Grid** (~5.5 km)  
- **Reach** (~27.5 km)  
- **Expanse** (~137 km)

**Usage rules:**

- Close detail: finger, hand, span ("The gap was three fingers wide").

def parse_chapter(chunk):
    # chunk contains lines with "- **POV:**" etc
    d={}
    for field in ["POV","Tide","Setting","Beat Method","Function","Word Count Target","Emotional Arc","Soulpulse"]:
        m=re.search(rf"- \*\*{re.escape(field)}:\*\* (.+)", chunk)
        if m: d[field]=m.group(1).strip()
    # title? already
    # capture "Summary" section: after "**Summary:**" until next blank line with "**Key Elements" etc
    sm=re.search(r"\*\*Summary:\*\*(.*?)(?:\n\*\*Key Elements|\n\*\*Soulpulse|\n\*\*Stage|\n\*\*Arc|\Z)", chunk, flags=re.S)
    if sm:
        d["Summary"]=sm.group(1).strip()
    # key elements bullets
    km=re.search(r"\*\*Key Elements:\*\*(.*?)(?:\n\*\*Soulpulse|\n\*\*Stage|\n\*\*Arc|\Z)", chunk, flags=re.S)
    if km:
        d["Key Elements"]=km.group(1).strip()
    # stage progression
    pm=re.search(r"\*\*Stage Progression:\*\*(.*?)(?:\n\n---|\Z)", chunk, flags=re.S)
    if pm: d["Stage Progression"]=pm.group(1).strip()
    return d

def parse_segment(seg_text):
    # title line first
    first_line = seg_text.splitlines()[0]
    # Extract number/title
    if "CHAPTER" in first_line:
        m=re.search(r'CHAPTER (\d+): "([^"]+)"', first_line)
        num=int(m.group(1)) if m else None
        title=m.group(2) if m else None
        kind="CHAPTER"
    elif "INTERLUDE" in first_line:
        m=re.search(r'INTERLUDE ([IVX]+): "([^"]+)"', first_line)
        num=m.group(1) if m else None
        title=m.group(2) if m else None
        kind="INTERLUDE"
    else:
        kind="OTHER"; num=None; title=None
    # fields
    fields={}
    for field in ["POV","Tide","Setting","Beat Method","Saga Beat","Type","Word Count","Word Count Target","Target Word Count","Status","Emotional Intensity","Function","Arc Pattern","Purpose"]:
        fields[field]=get_field(seg_text, field)
    keybeats=extract_key_beats(seg_text)
    soulpulse=extract_soulpulse(seg_text)
    stage=extract_stage(seg_text)
    summary= re.search(r"\*\*Summary:\*\*(.*?)(?:\n\n\*\*Key Elements:|\n\*\*Key Beats:|\n\*\*Purpose:|\Z)", seg_text, flags=re.S)
    # But summary block sometimes empty, key beats etc
    return dict(kind=kind,num=num,title=title,fields=fields,keybeats=keybeats,soulpulse=soulpulse,stage=stage,text=seg_text)

### 3.2 Distance Units

From smallest to largest (body to continent):

- **Finger** (~2 cm)  
- **Hand** (~10 cm)  
- **Span** (~23 cm)  
- **Forearm** (~45 cm)  
- **Stride** (~90 cm)  
- **Bar** (~1.8 m)  
- **Rod** (~9 m)  
- **Cable** (~45 m)  
- **Vein-span** (~225 m)  
- **Circuit** (~1.1 km)  
- **Grid** (~5.5 km)  
- **Reach** (~27.5 km)  
- **Expanse** (~137 km)

**Usage rules:**

- Close detail: finger, hand, span ("The gap was three fingers wide").  
- Room / workshop: stride, bar ("The arch stood a bar and a half tall").  
- Street / block / infrastructure: rod, cable, vein-span.  
- Neighborhood / district / city: circuit, grid.  
- Regional / continental: reach, expanse.

Avoid metric/imperial in narrative. If you need a "translation," keep it in out-of-story material (bibles, notes), not in the prose.
