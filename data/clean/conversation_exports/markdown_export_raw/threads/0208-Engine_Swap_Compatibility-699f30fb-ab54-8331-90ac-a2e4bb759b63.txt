## 1. What Engines Are We Talking?

Common options:

- **2.0T Theta II (turbo)**
- **2.4L MPI (non-GDI, older trims)**

If both cars are:
- Same displacement  
- Same fuel system (GDI vs MPI)  
- Both turbo or both N/A  

Then you’re in business.

If not? It’s not a “swap.” It’s a wiring, ECU, and headache festival.

---

## 2. The Big Compatibility Checks

### ✅ Engine Code Match
Physically similar doesn’t mean identical. Check:
- Engine code stamped on block
- VIN engine code
- Build tag

Hyundai/Kia loves mid-cycle sensor changes.

---

### ⚡ Wiring Harness & Sensors
2013 → 2016 means:
- Different cam/crank sensors possible
- Different knock sensor revisions
- Different throttle body plug
- Different injector connectors

Best move:
- Use the **Sonata’s engine harness and sensors**
- Swap over Optima long block only

Do NOT try mixing harnesses unless you like chasing ghosts.

---

### 🧠 ECU Compatibility
Keep the Sonata ECU with Sonata wiring.

If you try running a 2013 Optima ECU in a 2016 Sonata:
- Immobilizer issues
- CAN bus mismatch
- Cluster freakout

Not worth it.

---

### 🔧 Mounts & Brackets
Usually interchangeable, but check:
- Engine mount brackets
- AC compressor bracket
- Power steering pump style (if hydraulic)
- Alternator plug orientation

Sometimes the block is the same but brackets differ slightly.

---

### 🛠 Intake / Exhaust Differences
Even with same engine:
- Exhaust manifold may differ
- Downpipe flange clocking can change
- Intake manifold sensor provisions can vary

Swap manifolds from the Sonata if needed.

---

## 3. Known Theta II Reality Check

If it’s the **2.4 GDI Theta II**, you already know:
- Rod bearing issues
- Oil consumption
- Knock sensor detection updates
- Recall/extended warranty programs

If you're swapping because one spun a bearing — inspect the replacement hard:
- Drop oil pan
- Check for glitter
- Plastigauge rods if you want to sleep at night

## 4. Smart Way to Do This Swap

If engines match:

**Use from Sonata:**
- Wiring harness
- Intake manifold
- Exhaust manifold
- Sensors
- HPFP
- Injectors (if compatible)

**Use from Optima:**
- Long block only (block + head)

That avoids 90% of compatibility drama.
