### 2) Continuity checks (put under `/scripts/`)
**`/scripts/validate_lore.py`**
```python
import sys, json, yaml, jsonschema, pathlib

root = pathlib.Path(__file__).resolve().parents[1]  # repo root
data_dir = root / "data"

def load_yaml(p): return yaml.safe_load(p.read_text(encoding="utf-8"))

def validate(data_path, schema_path):
    data = load_yaml(data_path)
    schema = load_yaml(schema_path)
    jsonschema.validate(data, schema)

def continuity_guards(novellas, characters):
    # Example guard: dead characters can't have new beats unless spectral
    char_status = {c["id"]: c["status"] for c in characters["characters"]}
    spectral = {c["id"] for c in characters["characters"] if c["status"]=="spectral"}
    for n in novellas["novellas"]:
        mentioned = set()
        for b in n.get("beats", []):
            for r in b.get("references", []):
                if r.startswith("CHAR:"):
                    mentioned.add(r.split(":",1)[1])
        for cid in mentioned:
            if char_status.get(cid) == "dead" and cid not in spectral:
                raise AssertionError(f"Continuity: dead character {cid} in {n['id']} without spectral status")

if __name__ == "__main__":
    validate(data_dir/"novellas.yaml", data_dir/"novellas.schema.yaml")
    validate(data_dir/"characters.yaml", data_dir/"characters.schema.yaml")
    continuity_guards(
        load_yaml(data_dir/"novellas.yaml"),
        load_yaml(data_dir/"characters.yaml"),
    )
    print("Lore validation passed.")
```

3. It opens a PR with everything wired: data layer, validators, MkDocs, pre-commit, link check, and the steady-state CI. Merge. Then enable **Settings → Pages → GitHub Actions** to publish docs.

### Notes (so you don’t get surprised)
- The workflow uses the built-in `GITHUB_TOKEN` to create the PR. No extra secrets needed.
- It also fixes that cursed `.../ 03_Protocols` folder if it exists.
- Git LFS is configured, but you still need to run `git lfs install` locally if you’re pushing big binaries later; LFS pointers are harmless if you never use them.

If you want this to **auto-merge** on green CI, I can add a step to set a branch protection rule and use an automerge action.
